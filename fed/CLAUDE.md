# CLAUDE.md - Fed Project Rules

## Component Rules

Follow the conventions defined in [`packages/aqua-style/README.md`](./packages/aqua-style/README.md):

- Component Structure
- Import Patterns
- Component Definition (3-file pattern)
- Root Class Rule
- CSS Class Grouping

## Before Commit

When source files (`.ts`, `.tsx`, `.css`, etc.) were changed, run format and lint:

```bash
cd fed
pnpm format
pnpm lint
```
